### 项目说明
本项目是一个快速启动的dubbo脚手架，基于官方文档，以mvp模式支持了基本的注册和网络调用。

https://blog.csdn.net/qq_27834905/article/details/107790140  
https://my.oschina.net/yunduansing/blog/4467033
